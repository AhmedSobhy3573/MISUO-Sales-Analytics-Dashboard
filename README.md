# MISUO Sales Analytics Project

## Project Overview
The **MISUO Sales Analytics Project** is an end-to-end data analytics project developed as part of the *Data Analytics Diploma*. It demonstrates a complete analytics workflow starting from raw data handling, through data cleaning and analysis, and ending with an interactive business dashboard and professional documentation.

This repository is structured following **real-world data analytics best practices** and is suitable for academic evaluation, portfolio presentation, and freelance or professional review.

## 01_Data
This directory contains all datasets used throughout the project lifecycle.

### Raw_Data
Original, unprocessed data files as received from the source.

- **MISUO_Sales_Raw.xlsx**  
  Original raw sales dataset.  
  **Note:** This file is **not public** and is included for internal analysis purposes only.

### Cleaned_Data
Final, cleaned, and analysis-ready datasets.

- **MISUO_Sales_Cleaned.xlsx**  
  Cleaned version of the sales data after handling missing values, data validation, formatting, and feature derivation.

### Data_Dictionary
Contains documentation that explains each column in the dataset, including definitions, data types, and possible values to ensure data clarity and consistency.

---

## 02_Data_Cleaning
This directory documents the data cleaning and preprocessing phase.

- **Cleaning_Steps_Documentation.pdf**  
  Details all data cleaning steps applied to the raw data, including:
  - Missing value treatment
  - Data consistency checks
  - Date and numeric formatting
  - Creation of calculated and derived columns

---

## 03_Data_Analysis
Contains analytical Excel files built using Pivot Tables, Pivot Charts, and calculated KPIs. Each file focuses on a specific business perspective.

- **Sales_Performance.xlsx** – Overall sales performance and core KPIs
- **Product_Analysis.xlsx** – Product-level performance and contribution analysis
- **Customer_Analysis.xlsx** – Customer behavior and segmentation analysis
- **Platform_Analysis.xlsx** – Sales performance across different sales platforms
- **Area_Analysis.xlsx** – Geographic and regional sales distribution
- **Delivery_Analysis.xlsx** – Delivery performance, delays, and on-time metrics

---

## 04_Dashboard
Contains the final interactive dashboard and supporting visual assets.

- **MISUO_Sales_Dashboard.xlsx**
